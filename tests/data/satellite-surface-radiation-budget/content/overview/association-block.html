The table below aims to help the user to navigate among the naming of radiation related variables in different products. In the four left-hand columns of the table we present the relationship between the names of variables as they can be found in the files provided by satellite measurements (CLARA and CCI family), climate reanalysis (ERA5) and climate projections (CMIP5). The four right-hand columns provide the names used in the CDS download forms (which sometimes do not match the names in the files). The variable names differ, but they point to the same physical quantities.
For the data provided in this catalogue entry (CLARA and CCI family data), the CDS Catalogue uses the same variable names as the corresponding long names in the netCDF files.
